## この論文の概要

腎陰虚証に用いられる漢方処方 **左帰飲（Jwagwieum／Joa-Gui Em、中国名: 坐帰飲 Zuoguiyin、以下 JGE）** の品質管理を目的に、**9種の指標成分を HPLC–PDA で同時定量する分析法を開発・バリデーション** した論文です（韓国・韓医学研究院, KIOM）。JGE は構成生薬が6種ある一方で品質管理に関する報告が少なく、規格設定の基礎データを整備したのが主眼です。直線性・LOD/LOQ・回収率・精度・システム適合性を検証し、実試料3ロットへ適用しています。

## 背景・目的

- JGE は **熟地黄（Rehmannia glutinosa）・山薬（Dioscorea japonica）・枸杞子（Lycium chinense）・山茱萸（Cornus officinalis）・茯苓（Poria cocos）・炙甘草（Glycyrrhiza uralensis）** の6生薬からなる処方。
- 効能（腎保護など）の報告はあるが、**バッチ間の一貫性を担保する品質管理法（標準化）の報告が乏しい**。
- そこで、薬局方（韓国KP・中国CP）の規格マーカーや既報の主要活性成分を踏まえ、**1法で複数成分を押さえる同時定量法**を整備することを目的とした。

## 指標成分（マーカー9成分）

由来生薬ごとに選定。薬局方マーカー（5-HMF, グリチルリチン, ロガニン, モロニサイド 等）も含む。

| No. | 成分 | 主な由来生薬 |
| --- | --- | --- |
| 1 | gallic acid（没食子酸） | 山茱萸 |
| 2 | 5-(hydroxymethyl)furfural（5-HMF） | 熟地黄 |
| 3 | morroniside（モロニサイド） | 山茱萸 |
| 4 | loganin（ロガニン） | 山茱萸 |
| 5 | liquiritin apioside | 甘草 |
| 6 | liquiritin（リクイリチン） | 甘草 |
| 7 | ononin（オノニン） | 甘草 |
| 8 | glycyrrhizin（グリチルリチン） | 甘草 |
| 9 | allantoin（アラントイン） | 山薬 |

> 補足: 山薬のアラントインと枸杞子のベタインは当初候補だったが、**ベタインはUV発色団がなく保持も短いため除外**、**アラントインはC18で検出されず別カラム（アミノ系）で分析**。実務上「1カラムで全部入り」にはならない点に注意。

## 方法（分析条件の要点）

最適化はカラム→移動相添加酸→カラム温度の3段階で実施。

- **本体8成分**: SunFire™ C18（250 mm × 4.6 mm, 5 µm）、移動相 = 水–アセトニトリル（0.1% ギ酸添加）グラジエント、**カラム温度 40 ℃**、PDA検出（各成分の極大吸収波長で定量: 210〜280 nm）。
- **アラントイン（1成分）**: Luna NH2 カラムで別途、イソクラティック分析。
- 試料調製: 凍結乾燥JGE 100 mg に 70% メタノール 10 mL、超音波抽出60分、0.2 µm 濾過。

実務的に効く知見:
- **カラム選定が分離の肝**。liquiritin apioside と liquiritin の分離度が列で大きく変動（SunFire 2.26 / Capcell 1.84 / Kinetex 1.14 / Hypersil GOLD 0.99）。SunFire を採用。
- 添加酸（ギ酸/TFA/リン酸/酢酸）では分離度に大差なし → 汎用性の高い **ギ酸** を採用。
- 温度依存: **30 ℃では没食子酸が未知ピークと重なり**、**50 ℃では没食子酸・5-HMFのテーリングとロガニンの重なり** → **40 ℃が最適**。

## 主な結果（バリデーション）

| 項目 | 結果 |
| --- | --- |
| 直線性 r² | ≥ 0.9980（良好） |
| LOD | 0.003–0.071 µg/mL（8成分）。※アラントインは別法で 0.764 µg/mL |
| LOQ | 0.010–0.216 µg/mL（8成分）。※アラントインは 2.315 µg/mL |
| 回収率（accuracy） | 96.36–106.95%（RSD ≤ 2.50%） |
| 精度（intraday/interday） | RSD < 1.20% |
| 安定性（72時間） | 97.14–105.24%（RSD ≤ 2.06%） |
| システム適合性 | k ≥ 1.16, α ≥ 1.03, Rs ≥ 2.25, N ≥ 16,566, S ≤ 1.19 |

実試料（凍結乾燥JGE 3ロット）での含量は **0.15–3.69 mg/g**。**グリチルリチンが最多（3.67–3.69 mg/g）** で、3ロット間のばらつきも小さく、処方の管理マーカーとして有用と示された。

## 実務へのインパクト・示唆

- **規格設定の出発点として使える**: 1回の HPLC–PDA 分析（＋アラントインは別カラム）で、由来生薬の異なる複数成分を一括して押さえられる。多成分処方の同等性・恒常性評価に直結。
- **甘草系成分（特にグリチルリチン）が定量的に安定** → 主力マーカー候補。逆に微量成分（ononin, liquiritin 等 0.15 mg/g 級）は LOQ との余裕を確認のうえ規格化したい。
- **条件のロバストネスに注意点が明示**: カラム銘柄差・温度（40 ℃厳守）が分離に効く。**他施設・他カラムへ展開する際は分離度（特に liquiritin apioside/liquiritin）と没食子酸まわりの妨害ピークを再確認**するのが安全。
- PDAなので**汎用機で再現しやすい**（MS不要）。ルーチンQCへの実装ハードルが低い。

## 限界・留意点

- アラントイン・ベタインのように **1法で全成分を網羅できない**（別カラム/対象外）。運用上は2系統の分析になる。
- バリデーションは **ICH Q2B 準拠** だが、頑健性（ロバストネス）の網羅的検討やロット数（実試料3）は限定的。原料産地・加工差を含む実生産ロットでの追検証が望ましい。
- 含量・効能の関連（薬物動態・生物学的利用能）は本研究の範囲外（著者も今後の課題と明記）。

## ひとことメモ

- 自社で類似処方（六味丸系・金匱腎気丸系など）のQCを組むなら、本法の **マーカー選定ロジック（薬局方収載＋由来生薬代表成分＋分離可能性）** がそのまま流用できる。
- 移植時の最初のチェックは「**40 ℃**」「**SunFire 相当のC18**」「**liquiritin apioside/liquiritin の分離度 >1.5**」「**没食子酸の妨害ピーク**」。ここが合えば再現性は高そう。
